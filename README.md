# 🛡️ Perplexity Spaces Backup & Restore

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![Firefox](https://img.shields.io/badge/browser-Firefox-orange)
![Manifest](https://img.shields.io/badge/manifest-V3-green)
![Tech](https://img.shields.io/badge/stack-WXT_|_Preact_|_TS-61dafb)

Надежное расширение для Firefox, позволяющее экспортировать все ваши Perplexity Spaces в JSON и восстанавливать их в другом аккаунте. Включает обход лимитов (Rate Limiting) и встроенный менеджер коллекций.

---

## 🚀 Возможности

- **Полный экспорт**: Сохраняет названия, эмодзи, инструкции (системные промпты) и описания.
- **Умный импорт**: Автоматическое определение дубликатов — расширение помечает спейсы, которые уже есть в аккаунте.
- **Обход лимитов**: Автоматическая очередь (1 спейс в минуту) с перезагрузкой страницы для сброса сессии Perplexity.
- **Менеджер спейсов**: Просмотр и удаление текущих коллекций прямо из расширения.
- **JSON формат**: Гарантирует корректное сохранение спецсимволов и вложенного кода в промптах.

## 🛠️ Разработка

### Требования
- **Node.js** (v18+)
- **pnpm** (рекомендуется)

### Быстрый старт
```bash
# Установка зависимостей
pnpm install

# Запуск в режиме разработки (откроет Firefox с расширением)
pnpm dev

# Проверка типов и линтинг
pnpm typecheck
pnpm lint
```

## 📦 Сборка и установка

### Как запаковать в .zip
```bash
pnpm pack:firefox
```
Архив появится в папке `.output/`.

### Ручная установка в Firefox
1. Перейдите по адресу `about:debugging#/runtime/this-firefox`.
2. Нажмите **"Load Temporary Add-on..."**.
3. Выберите `manifest.json` в папке `.output/firefox-mv3/` или созданный `.zip` архив.

## 📖 Как пользоваться

### 1. Экспорт (Бэкап)
1. Откройте **perplexity.ai** и авторизуйтесь.
2. Откройте расширение и нажмите **"Export to .json"**.
3. Сохраните файл. Это ваш слепок всех Spaces.

### 2. Импорт (Восстановление)
1. Нажмите **"🚀 Open Full Page Importer"** в расширении (это предотвратит закрытие окна при выборе файла).
2. Нажмите **"Select .json file"** и выберите ваш бэкап.
3. В появившемся списке:
   - **Check**: Отметьте те спейсы, которые хотите создать.
   - **Manual**: Используйте кнопку **"Create"** напротив спейса для мгновенного создания.
   - **Auto**: Нажмите **"Restore Selected"** для запуска очереди (рекомендуется для >5 спейсов).
4. **Важно**: При автоматическом восстановлении страница Perplexity будет периодически обновляться — это необходимо для обхода защиты от спама.

---

## ⚠️ Важные замечания
- Расширение работает через внутреннее API Perplexity (Content Script). 
- Необходимо держать хотя бы одну вкладку с открытым **perplexity.ai** во время работы.
- Режим **1/min** является самым безопасным для предотвращения временной блокировки вашего аккаунта.
